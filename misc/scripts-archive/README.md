# Archived one-off scripts

These are one-shot generators/parsers, not part of the build or `package.json`. They were
used to produce assets now committed under `public/`, or were exploratory parsers. Archived
here (out of the active `scripts/` tree) but kept in git for recoverability.

Still-live scripts kept in `scripts/`: demo-scraper.sh, engine-smoke.ts, run_campaign.ts,
and the two provenance-referenced generators gen_marketing_images.mjs + gen_brand_images.mjs.
